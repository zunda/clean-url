# clean-url
A web app that removes unwanted components from URLs.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

The artifacts will be in `dist/` directory.

### Lints and fixes files
```
yarn lint
```
